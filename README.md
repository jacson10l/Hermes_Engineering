# Hermes AI
## AI-Native Context Engineering Runtime

Hermes AI is a lightweight runtime for long-term AI coding collaboration.

Instead of stuffing massive prompts into LLMs, Hermes uses:

- layered memory
- retrieval-oriented context loading
- modular skills
- summary-first memory
- token-efficient runtime architecture

## Core Concepts

- Runtime Layer
- Retrieval Layer
- Skills Layer
- Agent Layer
- Archive Layer

## Philosophy

Hermes is not a giant prompt.

Hermes is:

> AI Context Operating System.

## Quick Start

```bash
git clone https://github.com/jacson10l/Hermes_Engineering
cd Hermes_Engineering
bash install.sh my-project
```

Then tell your AI:

```text
Initialize this project using Hermes Runtime.
```
